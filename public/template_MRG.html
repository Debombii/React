1124-0823

6/11/2024

Version 1.1.0.1

Funcionalidad de modificación de logs: Se ha añadido la capacidad de modificar logs específicos mediante su ID. Esto permite cambiar tanto el contenido como el título de un log sin afectar otros elementos de la página.

Correcciones:

Corrección en la actualización del id en el título: Se ha corregido un error en la funcionalidad de actualización del título de los logs. Anteriormente, al modificar el contenido de un log, se añadía el id en el contenido de manera incorrecta, ahora se reemplaza adecuadamente el id del <h3> con el nuevo título.

No modificación del <h2>: Se ha asegurado que al modificar un log, el elemento <h2> que contiene el identificador del log no se vea alterado. Esto es útil cuando solo se desea cambiar el contenido del <h3 class="titulo"> y no se debe tocar el encabezado principal de la versión.
