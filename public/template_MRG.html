Version 1.1.0.1

Notas de versión - v1.2.0 (6 de Noviembre de 2024)

Nuevas características:

  • Funcionalidad de modificación de logs: Se ha añadido la capacidad de modificar logs específicos mediante su ID. Esto permite cambiar tanto el contenido como el título de un log sin afectar otros elementos de la página.

    • Cambio en el título (<h3 class="titulo">): Se puede modificar el texto dentro de los <h3 class="titulo"> correspondientes a un log.
    • Cambio en el contenido: También es posible modificar el contenido detallado del log que está dentro de las etiquetas <h3 class="titular">.

Correcciones:

  • Corrección en la actualización del id en el título: Se ha corregido un error en la funcionalidad de actualización del título de los logs. Anteriormente, al modificar el contenido de un log, se añadía el id en el contenido de manera incorrecta, ahora se reemplaza adecuadamente el id del <h3> con el nuevo título.

  • No modificación del <h2>: Se ha asegurado que al modificar un log, el elemento <h2> que contiene el identificador del log no se vea alterado. Esto es útil cuando solo se desea cambiar el contenido del <h3 class="titulo"> y no se debe tocar el encabezado principal de la versión.

Mejoras de rendimiento:

  • Optimización en la búsqueda y modificación de logs: El proceso de búsqueda y modificación de logs ha sido optimizado, reduciendo el tiempo de respuesta cuando se modifican múltiples logs en un solo archivo.

Notas adicionales:

  • Ahora se puede realizar un seguimiento de cada log modificado a través del sistema de logs, permitiendo una mejor trazabilidad de las modificaciones realizadas.
  • Asegúrese de que el formato del log que se envía para modificación cumpla con los requisitos de identificación (id) y contenido.
